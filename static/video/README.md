# Presentation Video Directory

Place your recorded `presentation.mp4` video presentation file in this folder. 

Once placed:
1. It will be served at `/static/video/presentation.mp4`.
2. It will automatically load and play in the video section of the website homepage!

### Recommended Video Specs:
- **Format**: `.mp4`
- **Video Codec**: H.264 (standard for web players)
- **Resolution**: 1080p or 720p (landscape format)
